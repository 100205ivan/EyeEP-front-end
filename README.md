# EyeEP-front-end

Front-end prototype and UI modules for the EyeEP intelligent IEP support system.

EyeEP 智慧化 IEP 輔助系統的前端原型與 UI 模組。

---

# 專案介紹

EyeEP 是一套智慧化 IEP（Individualized Education Program）輔助系統。

本專案目前主要為：

- 前端 UI 展示
- 頁面互動原型
- 系統畫面設計
- 模組化頁面開發

目前以前端展示與功能原型為主，後續將整合：

- 後端系統
- AI 情緒辨識
- IEP 模板管理
- 學生資料管理
- 教師後台功能

---

# 目前功能模組

## EyeEP-login

目前已完成：

- 登入畫面 UI
- 響應式版面
- 動畫切換效果
- 基本前端互動

使用技術：

- HTML
- CSS
- JavaScript
- React JSX（部分）

# 預計開發模組

## 台灣互動地圖

功能：

- 點擊縣市
- 縣市 hover 效果
- 學校類型選擇
- IEP 模板導向

---

## Dashboard

功能：

- 教師首頁
- 學生資料總覽
- IEP 狀態管理
- 情緒分析結果顯示

---

## IEP 模板系統

功能：

- 國小模板
- 國中模板
- 高中模板
- 不同縣市模板

---

## AI 情緒分析模組

功能：

- 影像辨識
- 語音分析
- 情緒偵測
- 課堂行為分析
